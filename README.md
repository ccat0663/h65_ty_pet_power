#### 简介

- 使用17h65 ble芯片、涂鸦wbr3和培正电量模块做成的三路WiFi电表

- 通过配置可以实时监控家中用电情况

- 可以设置报警功率

- 同时监控三路电源

### 注意：项目涉及强电，注意安全，发生后果与本人无关
